Codebook for GPT as Data Generator Data

The purpose of this document is to offer guidance for interpreting the variables from the publicly posted data from Lehr et al.’s GPT as Data Generator Studies.  This codebook references four different data sets, corresponding to four different gender stereotypes that were studied:

	•	GPT as Data Generator A – gender good-bad 20230227
	•	GPT as Data Generator B – gender art-science 20230227
	•	GPT as Data Generator C – gender home-work 20230227
	•	GPT as Data Generator D – gender reading-math 20230227

For each of these data sets, the following variables are included.

Random ID – This column contains a unique ID for ChatGPT’s responses to a particular word dyad.  The order of dyads (and thus this ID) was randomized.

Gender Word – This column contains the specific word related to the target categories (male or female), which was presented to GPT as part of each dyad.

Stereotype Word – This column contains the specific word related to the attribute categories (varied depending on the data set), which was presented to ChatGPT as part of each dyad.

Target – This column contains the target category (male vs. female) represented by the Gender Word presented to ChatGPT as part of each dyad.

Attribute – This column contains the attribute category (good vs. bad, art vs. science, home vs. work, or reading vs. math) represented by the Stereotype Word presented to ChatGPT as part of each Dyad.

GPT-3.5 Response – This column contains the numerical rating (from 1.0 to 10.0) generated by GPT-3.5 to estimate the degree of cultural association between the words in each dyad.

GPT-4 Response – This column contains the numerical rating (from 1.0 to 10.0) generated by GPT-4 to estimate the degree of cultural association between the words in each dyad.
